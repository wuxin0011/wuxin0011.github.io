关键字段信息

  • id 主键id
  • name 仓库名称
  • html_url 仓库地址
  • description 仓库描述
  • stargazers_count star 数量
  • owner.avatar_url 头像

有了这几个关键信息就够了

# 测试

点击这里即可测试

  • 第一个输入框为过滤字段信息
  • 第二个为选择github star 项目是排序是降序还是升序，默认不排序
  • 第三个输入github 用户名,默认为我的github名wuxin0011

说明:用户名不修改情况下是不出现搜索框的，搜索结果是缓存到本地中，减少请求接口次数
